# Student Management System (C++)

This is a simple console-based student management system that I developed to practice C++ and basic data structures.

## What this project does
- Add new students
- List all students
- Search students by ID
- Delete students

## Why I built this
I created this project to improve my understanding of C++ programming, especially working with vectors, classes, and basic control structures.

## Technologies
- C++
- STL (vector)
- Object-Oriented Programming

## Notes
This project is focused on learning fundamentals, so it runs in a console environment without a graphical interface.
